O que são clientes?

Conceito de Client: É, basicamente, aquele que consome a informação de algum servidor.

O que são servidores?

Servidores são a parte responsavel por receber a requisição do lado do cliente e retornar para ele o que ele está requisitando.

Linguagem de programação

É uma forma de se escrever regras e instruções para o computador.

Sobre tags

Tags são elementos que definem a estrutura e conteudo de uma pagina web.

  1. html: Define o início e o fim de um documento HTML.
  2. head: Contém metadados, como o título da página e links para folhas de estilo.
  3. title: Especifica o título da página, que aparece na aba do navegador.
  4. body: Contém o conteúdo visível da página, como texto, imagens e links.
  5. h1, h2, ... h6: Usadas para definir cabeçalhos, com h1 sendo o mais importante.
  6. p: Define um parágrafo.
  7. a: Define um hyperlink.
  8. img: Insere uma imagem na página.

Sobre Atributos

Atributos nas tags são propriedades, basicamente

Atributos ex: id=”tt”, style=”ss”

Texto em HTML

  • h1, h2...h6: São para titulos, subtitulos e assim por diante.
  • p: são paragrafos.
  • Blockquote: Serve para fazer uma citação.
  • U: Deixa o texto/palavra com uma linha abaixo.
  • I: Deixa o texto italico.
  • Mark: Deixa o texto com o efeito de marcador de texto.
  • Sup: Permite você colocar simbolos acima do texto.

Listas em HTML

Lista Ordenada (ol) (Tem numeros para cada item)

Lista Não Ordenada (ul) (Não tem numeros para cada item e sim marcadores)

Links em HTML

Se usa a tag "a" com o href para fazer o lind de uma pagina externa ou para uma pagina dentro do projeto.
